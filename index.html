Objetivo

Entender cómo funcionan las estructuras de control en PHP con ejemplos fáciles y mostrando el resultado esperado.

If

El if sirve para ejecutar un bloque solo si la condición es verdadera.

<?php
$t = 14;
if ($t < 20) {
    echo "14 es menor que 20, por lo tanto se ejecuta el código dentro del IF";
}
?>
Resultado esperado:
14 es menor que 20, por lo tanto se ejecuta el código dentro del IF

Switch

Switch permite elegir entre varias opciones según el valor de una variable.

<?php
$colorfav = "rosa";

switch ($colorfav) {
  case "rosa":
    echo "Tu color favorito es el rosa, un capo";
    break;
  case "azul":
    echo "Tu color favorito es el azul, tienes buenos gustos";
    break;
  case "verde":
    echo "Tu color favorito es el verde? Ahora somos hermanos";
    break;
  default:
    echo "No te gustan los mejores colores >:(";
}
?>
Resultado esperado (con $colorfav = "rosa"):
Tu color favorito es el rosa, un capo

While

While repite un bloque de código mientras la condición sea verdadera.

<?php  
$i = 1;

while ($i < 6) {
  echo $i;
  $i++;
} 
?>
Resultado esperado:
12345

For

For se usa cuando ya sabes cuántas veces quieres repetir algo.

<?php
for ($x = 0; $x <= 10; $x++) {
  echo "El número es: $x \n";
}
?>
Resultado esperado:
El número es: 0
El número es: 1
…
El número es: 10

Foreach

Foreach sirve para recorrer todos los elementos de un arreglo.

<?php
$colores = array("rosa", "verde", "azul", "amarillo"); 

foreach ($colores as $y) {
  echo "$y \n";
}
?>
Resultado esperado:
rosa
verde
azul
amarillo
